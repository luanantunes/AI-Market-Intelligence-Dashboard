# 🤖 AI Market Intelligence Dashboard

Solução de Business Intelligence (BI) desenvolvida para análise estratégica do ecossistema de Inteligência Artificial Generativa em 2024.

## 📊 Visão Geral
Este dashboard consolida dados de benchmarks técnicos, tendências de investimento global e ferramentas de cálculo de ROI para implementação de IA em escala corporativa.

## 🚀 Funcionalidades
- **Market Benchmarks**: Comparativo de performance entre os principais LLMs (GPT-4o, Claude 3.5, Gemini 1.5).
- **Análise de Investimento**: Visualização de adoção de IA por setores da indústria.
- **Calculadora de ROI**: Ferramenta interativa para estimativa de economia e ganho de produtividade mensal/anual.

## 🛠️ Stack Tecnológica
- **Python** (Lógica e Processamento)
- **Streamlit** (Framework de UI)
- **Plotly** (Engine de Gráficos Interativos)
- **Pandas** (Estruturação de Dados)
